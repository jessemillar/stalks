## Setup
You'll need to set the `SLACK_WEBHOOK`, `DATABASE_NAME`, `DATABASE_USERNAME`, `DATABASE_PASSWORD`, `DATABASE_PORT`, and `DATABASE_HOST` environment variables in order for database connections to function.
